Degré d'un polynôme

Soit P,Q∈𝕂[X], λ∈𝕂

Somme de polynôme
deg(P+Q) ≤ max{deg(P) + deg(Q)}
deg(P) ≠ deg(Q) ⇒ deg(P+Q) = max{deg(P) + deg(Q)}

Produit
si λ≠ 0, deg(λP) = λdeg(P)
deg(PQ) = deg(P) + deg(Q)
n∈ℕ*. deg(Pn) = n·deg(P)

Composition
Si Q n'est pas constant (deg(Q) ≤ 0) : deg(P∘Q) = deg(P) × deg(Q)

Dimension

E,F espaces vectoriels de dimension finie

Dimensions standards
dim(E×F) = dim(E) + dim(F)
dim(ℒ(E,F)) = dim(E) × dim(F)
dim(𝕂n) = n
dim(𝕂n[X]) = n+1
dim(ℳn,p(𝕂)) = n×p

Inclusion
F⊂E ⇒ dim(F) ≤ dim(E)
F⊂E, dim(F) = dim(E) ⇒ F = E

Somme
dim(E⊕F) = dim(E) + dim(F)
dim(E+F) = dim(E) + dim(F) − dim(E∩F) (Formule de Grassmann)
dim(F1+…Fp) ≤ dim(F1) + ⋯ + dim(Fp) (Égalité ssi somme directe)

Rang

de vecteurs
u1,…,up∈E
rg(u1,…,up) = dim(Vect(u1,…,up))
rg(u1,…,up) ≤ min(p,n)

d'une application linéaire
rg(f) = dim(Im(f))
g,h inversibles : rg(f∘g) = rg(f) = rg(h∘f)

d'une matrice
A = Matℬ(f) = (C1 ⋯ Cn)
rg(A) = rg(f) = rg(C1,⋯,Cn)
